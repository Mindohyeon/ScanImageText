# 이미지 텍스트 스캐너

## Description

이미지 or 선택되지 않는 글을 복사하려고 할 때 힘들었던 경험을 바탕으로 만들었습니다.

원하는 부분을 캡쳐해 앱으로 **드래그 & 드랍**하면 이미지에서 **텍스트를 추출합니다.**

추출된 텍스트는 **수정 & 복사**가 가능합니다.
